Stacks multiple tasks into a TaskContainer for hierarchical QP.

Parameters
AConstraint matrix (rows = total constraints, cols = variables)
buUpper bounds vector (size = total constraints)
blLower bounds vector (size = total constraints)
break_pointsVector of indices marking the end of each task in the stack
Returns
TaskContainer with each task as a GenericTask

Requirements:

  • A.rows() == bu.size() == bl.size()
  • break_points must be increasing and the last element equal to A.rows()
