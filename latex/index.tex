This project combines task-\/based hierarchies with advanced matrix decomposition techniques to resolve both equality and inequality constraints with exceptional numerical stability.

{\bfseries{Features include\+:}}
\begin{DoxyItemize}
\item Hierarchical Task Management\+: Aggregates multiple tasks with different priority levels.
\item Flexible Constraint Handling\+: Supports both equality and inequality constraints.
\item Robust Numerical Methods\+: Uses Cholesky decompositions and orthogonal decompositions.
\item Comprehensive Documentation\+: Built with Doxygen to offer full API insights and diagrams.
