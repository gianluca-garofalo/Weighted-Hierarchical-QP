\doxysubsection{Detailed Description}
Test driver for validating the Hierarchical-\/\+QP solver functionality. 

This program sets up multiple tasks with varying configurations, invokes the solver, and prints out the solution along with the active set details. It demonstrates the usage of both standalone and composite tasks.
