\doxysubsection{Detailed Description}
Main interface for the Weighted-\/\+Hierarchical-\/\+QP solver. 

This file defines the Hierarchical\+QP class which encapsulates the algorithm to solve prioritized quadratic programming problems. The solver aggregates multiple tasks and applies iterative refinements, handling both equality and inequality constraints.
