\doxysubsection{Detailed Description}
Declares the abstract Task interface and its derivatives used in the HQP solver. 

The file defines the base Task class and the templated Task\+Interface which allows passing of different parameter types to the run() method. Also included is the Sub\+Tasks class which aggregates multiple tasks into a composite unit.
